# 🌌 Stellar Atlas: Gaia DR3 Visualizer

**Stellar Atlas** — это передовое веб-приложение для интерактивного исследования Млечного Пути, базирующееся на данных миссии **Gaia Data Release 3 (DR3)**. Проект демонстрирует интеграцию научных астрономических каталогов с технологиями компьютерного зрения (Computer Vision) для создания бесконтактного интерфейса управления.

---

## 🔭 О проекте
В основе лежит визуализация звездного атласа, разработанная для образовательных и исследовательских целей. Приложение переносит массивы данных Европейского космического агентства (ESA) в интерактивную 3D-среду, доступную прямо в браузере.



## 🖐 Инновационное управление (Gesture Control)
Ключевая особенность проекта — отсутствие необходимости в клавиатуре или мыши. Управление происходит через камеру устройства с использованием алгоритмов распознавания жестов:

| Жест | Действие |
| :--- | :--- |
| ✊ **Кулак** | Свободное вращение 3D-сцены |
| 🤏 **Пинч** | Масштабирование (Zoom In/Out) |
| ✋ **Ладонь** | Панорамное перемещение (Panning) |
| ☝ **Палец** | Выбор объекта и просмотр карточки звезды |
| ✌ **Мир** | Сброс положения камеры до исходного |

---

## ⚙️ Технические характеристики

* **Data Source:** Использование официальных датасетов **Gaia DR3** (один из самых точных каталогов звезд в мире).
* **Rendering:** Оптимизированный рендеринг через HTML5 `<canvas>`, обеспечивающий плавность даже при отображении тысяч объектов.
* **Computer Vision:** Обработка видеопотока в реальном времени для детекции жестов (требует разрешения на использование камеры).
* **Performance:** Интегрированный счетчик FPS для мониторинга производительности в реальном времени.

---

## 📊 Информационный дашборд
При выборе звезды пользователь получает доступ к полным научным метаданным:
1.  **ID объекта:** Уникальный идентификатор Gaia DR3.
2.  **Спектральные характеристики:** Блеск (mag) и параметры светимости.
3.  **Пространственное положение:** Координаты в системе X, Y, Z.
4.  **Расстояние:** Дистанция до звезды, выраженная в парсеках (пк) или килопарсеках (кпк).
